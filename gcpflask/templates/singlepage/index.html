Monthly Counts For the Thing

Visualizing monthly counts for the Thing.

Welcome to Monthly Counts, a web interface that shows our compiled research results that aimed to derive counts for the Thing. For each month, we generate a random set of numbers using JavaScript, and create a plot for you to ponder over. You could also easily use a Python function to load or otherwise manipulate data for this view. Please reach out to RSE Services if you'd like some help!
